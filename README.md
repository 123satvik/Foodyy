# 🍔 Foodyy – Food Delivery Website (Frontend)

## 📌 Project Overview

Foodyy is a **basic and beginner-friendly food delivery website frontend** built using **React and Vite**.
The project focuses on creating a modern UI for browsing food items, viewing menus, and exploring a food delivery interface.

This project was built as a **practice project to learn React components, UI structure, and frontend development.**

---

## 🚀 Features

* Responsive food delivery website layout
* Food items displayed using reusable React components
* Clean and modern user interface
* Navigation between sections
* Beginner-friendly project structure

---

## 🛠️ Technologies Used

* **React.js**
* **Vite**
* **JavaScript (ES6)**
* **HTML5**
* **CSS3**

## ⚙️ Installation & Setup

1️⃣ Clone the repository

```
git clone https://github.com/123satvik/Foodyy.git
```

2️⃣ Navigate to the project folder

```
cd Foodyy
```

3️⃣ Install dependencies

```
npm install
```

4️⃣ Run the development server

```
npm run dev
```

Open the project in your browser at:

```
http://localhost:5173
```
